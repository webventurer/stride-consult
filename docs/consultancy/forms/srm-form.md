## Service

**Stride consultancy — agentic engineering enablement.**

A short, fixed-price engagement that teaches a technically-curious employee to build, ship, and maintain small internal apps and dashboards using AI coding tools — under the client's own infrastructure and governance.

## Service description

**Stride** is built around Anthropic's Claude Code (a terminal-based AI coding agent) and Linear (issue tracker), wrapped in a kanban workflow that produces auditable work — every change is a Linear issue, a git commit on a branch, and a pull request in the client's own [private GitHub organisation](/reference/git-hosting) (or approved equivalent: Gitea, GitLab CE, Bitbucket, Azure DevOps).

The engagement runs in three stages:

1. **Install and setup (2 days, on-site or remote).** Provision the toolchain on the user's Mac (Homebrew, Node.js, Python, git, Claude Code CLI), connect Stride to the client's Linear workspace via MCP, and ship one real internal app end to end by close of day 2.
2. **Weekly support (1 day per week for 3 weeks).** Review the user's independent work and correct anti-patterns early (atomic-commit discipline, kanban drift, premature optimisation).
3. **Ad-hoc support (optional, post-engagement).** Monthly check-ins tapering to on-demand.

**Deliverables.** One trained internal user able to build and ship their own apps unsupervised, demonstrated by at least one working internal app shipped end to end on the client's infrastructure during the engagement.

**Data and IP.** All code is owned by the client and lives in the client's GitHub org (or approved alternative). All work history lives in the client's Linear workspace. Webventurer hosts nothing. Under Anthropic's Commercial Terms (Claude Team or Enterprise), client code and prompts are contractually excluded from model training. An Anthropic Data Processing Addendum (DPA) is available from [Anthropic's Trust Center](https://trust.anthropic.com) for the client's DPO to sign directly with Anthropic — Webventurer is not party to it.

**Continuity.** Engagement is delivered by Mike Mindel. If the user leaves the client, the apps survive in-place — they run on the client's GitHub org (or approved alternative) and infrastructure, and a successor can pick them up with one day of hand-over.

**Pricing.** £5,000 ex VAT for Mike's consultancy time (the Stride workflow itself is complimentary), plus tooling (Claude subscription, GitHub, Linear) paid directly by the client. Detailed breakdown on the [How much?](/consultancy/how-much) page.
